# 0x01 Overview

Conda 是一个包管理器，最早只被用在 Python 中，用于解决在不同 Data Science 项目中依赖的问题。发展至今，Conda 已经不仅先于 Python, 其他编程语言也可以使用 Conda 来管理[^1]
