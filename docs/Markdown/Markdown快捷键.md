### #快捷键

ctrl + n 新建

ctrl + s 保存

ctrl + f 查找替换

ctrl + / 源码模式

ctrl + shift + l 左边导航栏

F8 专注模式

F9 打字机模式

### #标题

```markdown
# 一级标题 ctrl + 1

## 二级标题 ctrl + 2

### 三级标题 ctrl + 4

... 以此类推
```

### #文字格式

```markdown
<u>下划线</u>  ctrl + u

*斜体*  ctrl + i

**加粗**  ctrl + b

~~删除线~~  
```

效果显示:

<u>下划线</u>  ctrl + u

*斜体*  ctrl + i

**加粗**  ctrl + b

~~删除线~~  

### #链接图片

### #列表

```markdown
- order1 ctrl + shift + ]
- order2
  - order 2.1
  - order 2.2
1. order1  ctrl + shift + [
2. order2
```

效果显示:

- order1 
- order2 
  - order 2.1
  - order 2.2

1. order1
2. order2

### #HTML

```markdown
<font color="red" size=7 face="黑体">html代码</font>

<font face= "微软雅黑">我是微软雅黑</font>

<table><tr><td bgcolor=orange>背景色是：orange</td></tr></table>

```

效果显示:

<font color="red" size=7 face="黑体">html代码</font>

<font face= "微软雅黑">我是微软雅黑</font>

<table><tr><td bgcolor=orange>背景色是：orange</td></tr></table>
